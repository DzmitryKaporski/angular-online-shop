Getting started:

  1. Install JSON Server global `npm install -g json-server`
  2. Install json-server in the project `npm i json-server`
  3. Run json-server with command `json-server --watch db.json`. Navigate to `http://localhost:3000`.
  4. Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.
